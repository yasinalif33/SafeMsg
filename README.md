# 🛡️ SafeMsg Companion

SafeMsg Companion is a professional, high-performance web application designed for freelancers and agency teams to keep their Fiverr communications safe and compliant. It proactively scans raw text for restricted words such as contact information, off-platform payment requests, and academic fraud terms and instantly sanitizes them to prevent account warnings.

## ✨ Features

* **Real-Time Processing:** Zero-click live filtering. The text is sanitized the moment it is typed or pasted.
* **"God-Mode" Dictionary:** A deeply researched, extensive dictionary targeting Fiverr TOS algorithms, covering:
  * Prohibited contact methods (Skype, WhatsApp, Email, etc.)
  * Off-platform payment gateways (PayPal, Crypto, Stripe, etc.)
  * Academic fraud and homework-related keywords.
  * Review manipulation (e.g., "5 stars").
  * Gambling and NSFW terminology.
* **Smart Text Replacement:** Preserves exact text casing (e.g., "Review" becomes "Check", "REVIEW" becomes "CHECK") and intelligently auto-hyphenates complex flagged words to bypass automated flagging.
* **Premium UX/UI:** * Sleek, fixed-height workspace that never breaks, regardless of word count.
  * Custom micro-interaction "pop-in" animations for fixed words.
  * One-click copy and paste functionality.
* **Smart Dark Mode:** Seamlessly syncs with your operating system's default theme, with a manual override toggle that remembers your preference.

This application is built with pure, vanilla web technologies (HTML, CSS, JavaScript) and requires no build tools, package managers, or server-side rendering.

## 🛠️ Tech Stack

* **HTML5:** Semantic structure and crisp SVG icons.
* **CSS3:** Native CSS variables, Flexbox layout, smooth transition states, and custom keyframe animations.
* **Vanilla JavaScript:** Fast, lightweight DOM manipulation and RegEx-based text processing. No external dependencies.
